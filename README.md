# Magento Automation Tests

This project contains Selenium automation tests for Magento e-commerce website.

## How to Run
1. Clone this repository
2. Run: `mvn test`
3. Run: 'allure serve' --> for reports

## Test Cases
- Search functionality
- Add to cart
- Checkout process  
- Security testing

## Tech Stack
- Java 17
- Selenium WebDriver
- TestNG
- Maven
